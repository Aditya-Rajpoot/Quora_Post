# Quara Posts 

A full-stack **Quora-inspired post app** built with Node.js, Express, and EJS — created for learning RESTful API concepts.

## Features

- View all posts
- Create a new post
- View a single post in detail
- Edit a post
- Delete a post

## Tech Stack

- **Backend:** Node.js, Express.js
- **Templating:** EJS
- **Styling:** CSS
- **Packages:** uuid, method-override
